# RAWG Games Explorer

Скрипт для получения информации об играх в жанре Action через API сервиса RAWG. Выводит название, дату релиза, ссылку на страницу игры и список магазинов, где её можно купить.

## Как установить

1. Python3 должен быть уже установлен. Затем используйте `pip` (или pip3, если есть конфликт с Python2).

2. Установка виртуального окружения

```
python -m venv venv
```
И активируйте его:
* Windows:
```
venv\Scripts\activate
```
* Linux/MacOS:
```
source venv/bin/activate
```


3. Для установки зависимостей:

```
pip install -r requirements.txt
```

4. Для работы скрипта требуется API-ключ от сервиса [RAWG.io](https://rawg.io/):

* Зарегистрируйтесь на сайте [RAWG Developer Console](https://rawg.io/apidocs).
* Подайте заявку и получите бесплатный API-ключ.
* Создайте в корне проекта файл `.env` и добавьте в него строку:

```
RAWG_API_KEY=ваш_ключ_здесь
```

## Как запустить проект

Запустите скрипт можно из командной строки по команде:

```
python main.py
```

(Предполагается, что основной файл скрипта называется `main.py`. Если у вас другое имя — замените его.)

## Цель проекта

Код написан в образовательных целях для демонстрации работы с внешними API.
