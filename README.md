# RAWG Games Explorer

Скрипт для получения информации об играх в жанре Action через API сервиса RAWG. Выводит название, дату релиза, ссылку на страницу игры и список магазинов, где её можно купить.

## Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:

pip install -r requirements.txt

Для работы скрипта требуется API-ключ от сервиса RAWG.io:
- Зарегистрируйтесь на сайте RAWG Developer Console.
- Создайте новый проект и получите бесплатный API-ключ.
- Создайте в корне проекта файл .env и добавьте в него строку:

RAWG_API_KEY=ваш_ключ_здесь
- Пример содержимого файла .env:

RAWG_API_KEY=fde54e9c463f4b81ab76f82be0540a14

## Как запустить проект

Запустите скрипт из командной строки:

python main.py

(Предполагается, что основной файл скрипта называется main.py. Если у вас другое имя — замените его.)

## Цель проекта

Код написан в образовательных целях для демонстрации работы с внешними API.
